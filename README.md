💰 FinTrack – AI Powered Personal Finance Tracker

An AI-powered Personal Finance Tracker built using the MERN Stack that helps users manage their income, expenses, budgets, and financial insights through an intuitive dashboard.

---

## 🚀 Features

### 🔐 Authentication
- User Registration & Login
- JWT Authentication
- Protected Routes
- Secure Password Hashing

### 📊 Dashboard
- Total Income
- Total Expenses
- Savings Overview
- Recent Transactions
- Interactive Charts

### 💳 Transaction Management
- Add Transactions
- Edit Transactions
- Delete Transactions
- Search Transactions
- Filter by Category & Type
- Sort by Date & Amount
- Pagination

### 💵 Budget Management
- Set Monthly Budget
- Category-wise Budget Limits
- Budget Progress Tracking
- Remaining Budget Calculation
- Overspending Alerts

### 🤖 AI Insights
- Personalized Financial Recommendations
- Spending Pattern Analysis
- Budget Health Analysis
- Expense Prediction
- AI & Heuristic-Based Insights

### 📈 Reports
- Monthly Financial Reports
- Income vs Expense Analysis
- Category-wise Expense Breakdown

## ⚙️ Installation

### Clone Repository

```bash
git clone github.com/shettypavan486-boop/Fin-Track.git
```

### Go to Project

```bash
cd FinTrack-AI-Personal-Finance-Tracker
```

### Install Backend

```bash
cd server
npm install
```

### Install Frontend

```bash
cd ../client
npm install
```

---

## ▶️ Run the Project

### Start Backend

```bash
cd server
npm run dev
```

### Start Frontend

```bash
cd client
npm run dev
```
